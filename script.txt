TO-DO App

When i click on add it will add new Task, and
When i click on delete it will remove  Task

lets break down How things work.

1- 1st we create an empty array to store all Task.

2- fir add button event create karte hai
 jaise add pe click hoga hum input se task uthate hai , aur array mai push() karte hai
 aur har bar list ko dobara , render karte hai using rederTask() function.

 3- renderTask() function pure list ko clean karta hai, fir map() method se sub task ko render karte hai, EACH task ke sath ek delete button generate hoti hai.

 4- Delete button pe click kartre hi deleteTask() function run hota hai aur splice() method se task remove hota hai , aur list dobara render hoti hai.
